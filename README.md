**ProPlanner Ultimate v2** is a sleek, gamified, and fully responsive personal task management application. Built entirely within a **single HTML file**, it requires absolutely zero build steps, servers, or external databases. Just open it in your browser and start organizing your life!

## ‚ú® Description

ProPlanner is designed to make productivity rewarding. It combines robust task management with gamification elements like daily streaks, XP points, and user leveling. Whether you need a simple daily checklist, a monthly calendar overview, or a distraction-free Pomodoro timer for deep work, ProPlanner has you covered.

It uses local storage to keep your data private and secure on your device, with built-in tools to export and import your workspace seamlessly.

## üåü Key Features

### üìã Task Management
* **Rich Task Details**: Add descriptions, deadlines, priorities (Low, Medium, High), and custom categories.
* **Recurring Tasks**: Set tasks to repeat daily, weekly, or monthly.
* **Global Search**: Quickly find tasks using the smart global search bar (Shortcut: `Ctrl/Cmd + K`).
* **Overdue Tracking**: Instantly view and catch up on overdue tasks.

### üéÆ Gamification
* **Leveling System**: Earn XP by completing tasks (+20 XP) and focus sessions (+50 XP) to level up.
* **Activity Tracking**: Visual activity graph showing your task completion rate over the last 7 days.
* **Daily Streaks**: Maintain your login streak to build consistent productivity habits.

### üìÖ Calendar Views
* **Monthly View**: A beautiful grid overview of your month with task indicators.
* **Daily View**: Drill down into specific days to see exactly what needs to be done.

### ‚è±Ô∏è Focus Mode
* Built-in **25-minute Pomodoro timer** with a sleek circular progress animation.
* Link specific tasks to your focus sessions.
* Distraction-free, full-screen overlay.

### üé® Personalization & Settings
* **3 Beautiful Themes**: Light Mode, Dark Mode, and Midnight (Void) Mode.
* **Responsive Design**: App-like mobile experience with a bottom navigation bar, and a powerful desktop sidebar layout.
* **Data Portability**: Export your workspace as a `.json` backup and import it across devices.

üõ†Ô∏è Tech Stack
HTML5: Semantic structure.
Tailwind CSS (via CDN): Rapid, utility-first styling with custom configurations for themes and animations.
Vanilla JavaScript: State management, local storage interaction, DOM manipulation, and routing all handled without a framework.
üìÇ Project Structure
The entire application lives in one file, cleanly organized into:
<head>: Metadata, Tailwind Configuration, Custom CSS variables, and Keyframe animations.
<body>:
Semantic UI layout (Sidebar, Mobile Nav, Main Content Area).
Hidden Modals (Task Creation, Settings, Focus Mode Overlay).
<script>: The app object containing state, router, views, data management, and UI logic.
üí° Usage Tips
Desktop Search: Press Ctrl + K (or Cmd + K on Mac) anywhere in the app to immediately focus the search bar.
Backups: Since data is saved in your browser's Local Storage, remember to periodically go to Settings > Export Backup to save a .json copy of your tasks to your local hard drive.
Mobile Experience: Add the page to your Home Screen on iOS or Android for a native app-like experience without the browser address bar.
